# 📊 Workflow Diagrams

This directory contains workflow diagrams for the DOL-EO-Management system.

## 📁 Structure

- `*.mmd` - Mermaid source files (ignored by git)
- `images/` - Generated PNG diagram images
- `README.md` - This file

## 🔄 Regenerating Diagrams

To regenerate the diagrams, you can use the Mermaid Live Editor:

1. Go to [Mermaid Live Editor](https://mermaid.live/)
2. Copy the content from any `.mmd` file
3. Paste it into the editor
4. Export as PNG
5. Save to `images/` directory

## 📋 Available Diagrams

- **email-processing-workflow.mmd** - Email processing system flow
- **authentication-flow.mmd** - JWT authentication sequence
- **task-management-workflow.mmd** - Task lifecycle management
- **daily-update-aggregation.mmd** - Daily summary generation
- **system-architecture.mmd** - System component architecture
- **user-interaction-flow.mmd** - User role-based interactions

## 🎨 Diagram Types

- **Flowcharts** - Process flows and decision trees
- **Sequence Diagrams** - Authentication and API interactions
- **Architecture Diagrams** - System component relationships
